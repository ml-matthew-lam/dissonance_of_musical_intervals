# Modeling Sensory Dissonance of Musical Intervals
Exploration of models of musical consonance based on frequency ratio, beating &amp; roughness, including William Sethares’s model from [Tuning, Timbre, Spectrum, Scale (1999)](https://link.springer.com/book/10.1007/b138848).
This project is a synthesis of my work on my International Baccalaureate (IB) Diploma Extended Essay in Mathematics.
